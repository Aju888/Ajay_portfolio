# 🌐 Website Deep Guide

## Folder Structure
```
website-deep-guide/
├── index.html        ← Main page (yahi browser mein kholo)
├── css/
│   └── style.css     ← Saari styling
├── js/
│   └── script.js     ← Interactivity
└── README.md         ← Ye file
```

## Internet Zaruri Hai?
Haan — Google Fonts load hone ke liye internet chahiye.

## Kaise Run Karein?
1. Poora folder download karo / unzip karo
2. `index.html` double-click karo — browser mein khulega
3. Ya Netlify par upload karo (FREE hosting)

## Online Kaise Karein?
1. netlify.com par free account banao
2. Poora `website-deep-guide` folder drag & drop karo
3. Link milega: https://yourname.netlify.app ✅
